安装 Supabase Cron Postgres 模块来开始调度定期执行的任务。

<Tabs
  scrollable
  size="small"
  type="underlined"
  defaultActiveId="dashboard"
  queryGroup="database-method"
>
<TabPanel id="dashboard" label="仪表盘">

1. 在仪表盘的集成部分下，进入 [Cron Postgres 模块](/dashboard/project/_/integrations/cron/overview)。
2. 启用 `pg_cron` 扩展。

</TabPanel>
<TabPanel id="sql" label="SQL">

```sql
create extension pg_cron with schema pg_catalog;

grant usage on schema cron to postgres;
grant all privileges on all tables in schema cron to postgres;
```

</TabPanel>
</Tabs>

## 卸载

通过禁用 `pg_cron` 扩展来卸载 Supabase Cron：

```sql
drop extension if exists pg_cron;
```

<Admonition type="danger">

禁用 `pg_cron` 扩展将永久删除所有任务。

</Admonition>
