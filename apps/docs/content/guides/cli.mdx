您可以使用 Supabase CLI 在本地机器上运行整个 Supabase 技术栈，方法是运行 `supabase init` 然后 `supabase start`。要安装 CLI，请参阅[安装指南](/docs/guides/cli/getting-started#installing-the-supabase-cli)。

Supabase CLI 提供了工具来在本地开发您的项目、部署到 Supabase 平台、处理数据库迁移，以及直接从您的数据库架构生成类型。
