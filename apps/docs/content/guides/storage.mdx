Supabase Storage 使上传和提供任何大小的文件变得简单，为文件访问控制提供了强大的框架。

## 功能特性

您可以使用 Supabase Storage 存储图像、视频、文档和任何其他文件类型。通过全球 CDN 提供您的资产，减少来自全球 285 多个城市的延迟。Supabase Storage 包含内置的图像优化器，因此您可以即时调整和压缩媒体文件的大小。
