Supabase 是一个托管平台，它使入门变得非常简单，无需管理任何基础设施。托管平台配备了许多由 Supabase 管理的安全和合规控制。

# 合规性

Supabase 符合 SOC 2 Type 2 标准并定期接受审计。Supabase 的所有项目都受相同的合规控制集的管辖。
[SOC 2 合规指南](/docs/guides/security/soc-2-compliance)更详细地解释了 Supabase 的 SOC 2 责任和控制。

[HIPAA 合规指南](/docs/guides/security/hipaa-compliance)解释了 Supabase 的 HIPAA 责任。通过 HIPAA 附加组件可以获得处理电子受保护健康信息 (ePHI) 并需要 HIPAA 合规性的项目的额外[安全和合规控制](/docs/guides/deployment/shared-responsibility-model#managing-healthcare-data)。

# 平台配置

Supabase 提供的每个产品都配备了可定制的安全控制，这些安全控制有助于确保在 Supabase 上构建的应用程序安全、合规并能够抵御各种威胁。

[安全配置指南](/docs/guides/security/product-security)提供了配置各个产品的详细信息。
