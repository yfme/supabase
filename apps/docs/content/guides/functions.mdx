Edge 函数是服务器端 TypeScript 函数，分布在全球各地的边缘节点——靠近您的用户。它们可用于侦听 webhook 或将您的 Supabase 项目与第三方集成[如 Stripe](https://github.com/supabase/supabase/tree/master/examples/edge-functions/supabase/functions/stripe-webhooks)。Edge 函数使用 [Deno](https://deno.com) 开发，为您作为开发人员提供以下几个好处：

- 它是开源的。
- 它是可移植的。Supabase Edge 函数可以在本地运行，也可以在任何其他兼容 Deno 的平台上运行（包括自托管基础设施）。
- 它优先支持 TypeScript 并支持 WASM。
- Edge 函数全球分布，具有低延迟特性。
