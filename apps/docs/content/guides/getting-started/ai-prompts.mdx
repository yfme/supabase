我们精心策划了一系列提示，帮助您使用喜爱的 AI 驱动的 IDE 工具（如 Cursor 或 GitHub Copilot）来处理 Supabase。

## 使用方法

将提示复制到您仓库中的一个文件里。

使用 AI 工具的"包含文件"功能，在与 AI 助手聊天时包含该提示。例如，在 Cursor 中，将它们添加为 [项目规则](https://docs.cursor.com/context/rules-for-ai#project-rules-recommended)；在 GitHub Copilot 中，使用 `#<文件名>`；在 Zed 中，使用 `/file`。
