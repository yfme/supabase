<StepHikeCompact>

  <StepHikeCompact.Step step={1}>

    <StepHikeCompact.Details title="创建 Hono 应用程序">

    使用 CLI 从 Supabase Samples 引导 Hono 示例应用程序。

    </StepHikeCompact.Details>

    <StepHikeCompact.Code>

      ```bash name=Terminal
      npx supabase@latest bootstrap hono
      ```

    </StepHikeCompact.Code>

  </StepHikeCompact.Step>

  <StepHikeCompact.Step step={2}>
    <StepHikeCompact.Details title="安装 Supabase 客户端库">

    项目中的 `package.json` 文件包含必要的依赖项，包括 `@supabase/supabase-js` 和 `@supabase/ssr`，用于帮助服务器端认证。

    </StepHikeCompact.Details>

    <StepHikeCompact.Code>

      ```bash name=Terminal
      npm install
      ```

    </StepHikeCompact.Code>

  </StepHikeCompact.Step>

  <StepHikeCompact.Step step={3}>
    <StepHikeCompact.Details title="设置所需的环境变量">

    将 `.env.example` 文件复制到 `.env` 并用您的 Supabase 项目 URL 和匿名密钥更新值。

    最后，在认证设置中[启用匿名登录](https://supabase.com/dashboard/project/_/settings/auth)。

    <ProjectConfigVariables variable="url" />
    <ProjectConfigVariables variable="anonKey" />

    </StepHikeCompact.Details>
    <StepHikeCompact.Code>


      ```bash name=Terminal
      cp .env.example .env
      ```

    </StepHikeCompact.Code>

  </StepHikeCompact.Step>

  <StepHikeCompact.Step step={4}>
    <StepHikeCompact.Details title="启动应用程序">

    启动应用程序，前往 http://localhost:5173。

    了解 [服务器端认证](/docs/guides/auth/server-side/creating-a-client?queryGroups=framework&framework=hono) 如何与 Hono 协同工作。

    </StepHikeCompact.Details>

    <StepHikeCompact.Code>

      ```bash name=Terminal
      npm run dev
      ```

    </StepHikeCompact.Code>

  </StepHikeCompact.Step>
</StepHikeCompact>
