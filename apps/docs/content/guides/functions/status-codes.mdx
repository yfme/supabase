## 2XX 成功

Edge Function 成功响应

## 3XX 重定向

Edge Function 通过 `Response.redirect` 响应 [API 文档](https://developer.mozilla.org/en-US/docs/Web/API/Response/redirect_static)

## 4XX 客户端错误

### 401 未授权

如果 Edge Function 启用了 `验证 JWT` 选项，但请求使用了无效的 JWT。

### 404 未找到

请求的 Edge Function 未找到。

### 405 方法不允许

Edge Functions 仅支持这些 HTTP 方法：'POST'、'GET'、'PUT'、'PATCH'、'DELETE'、'OPTIONS'

## 5XX 服务器错误

### 500 内部服务器错误

Edge Function 抛出未捕获的异常（`WORKER_ERROR`）。检查 Edge Function 日志以找出原因。

### 503 服务不可用

Edge Function 启动失败（`BOOT_ERROR`）。检查 Edge Function 日志以找出原因。

### 504 网关超时

Edge Function 在[请求空闲超时](/docs/guides/functions/limits)之前没有响应。

### 546 资源限制（自定义错误代码）

由于资源限制，Edge Function 执行被停止（`WORKER_LIMIT`）。Edge Function 日志应提供超出了哪个[资源限制](/docs/guides/functions/limits)。
