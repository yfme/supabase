<Admonition type="caution">

请注意，由于 Dart 在开发 (WASM) 支持方面出现了多次重大变更，Dart Edge 项目目前未被积极维护。

</Admonition>

[Dart Edge](https://docs.dartedge.dev/) 是一个实验性项目，使您能够使用 Dart 编写 Supabase 边缘函数。它由 [Invertase](https://invertase.io/) 构建和维护。

有关如何设置和使用 Dart Edge 与 Supabase 的详细信息，请参阅 [Supabase 官方 Dart Edge 文档](https://invertase.docs.page/dart_edge/platform/supabase)。
