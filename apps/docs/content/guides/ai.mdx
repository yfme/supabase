Supabase 提供了一套使用 Postgres 和 pgvector 开发 AI 应用程序的开源工具包。使用 Supabase 客户端库来存储、索引并大规模查询您的向量嵌入。

该工具包包括：

- 使用 Postgres 和 pgvector 的[向量存储](/docs/guides/ai/vector-columns)和嵌入支持。
- 用于管理非结构化嵌入的 [Python 客户端](/docs/guides/ai/vecs-python-client)。
- 使用 Edge Functions 中的开源模型进行[嵌入生成](/docs/guides/ai/quickstarts/generate-text-embeddings)的流程。
- 用于管理结构化嵌入的[数据库迁移](/docs/guides/ai/examples/headless-vector-search#prepare-your-database)。
- 与所有流行的 AI 提供商集成，如 [OpenAI](/docs/guides/ai/examples/openai)、[Hugging Face](/docs/guides/ai/hugging-face)、[LangChain](/docs/guides/ai/langchain) 等。

## 搜索

您可以使用 Supabase 为您的应用构建不同类型的搜索功能，包括：

- [语义搜索](/docs/guides/ai/semantic-search)：按含义而非确切关键词搜索
- [关键词搜索](/docs/guides/ai/keyword-search)：按单词或短语搜索
- [混合搜索](/docs/guides/ai/hybrid-search)：结合语义搜索和关键词搜索
