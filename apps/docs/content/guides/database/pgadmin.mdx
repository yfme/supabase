[`pgAdmin`](https://www.pgadmin.org/) 是一个用于管理 Postgres 数据库的 GUI 工具。您可以通过 SSL 使用它连接到您的数据库。

  <StepHikeCompact.Step step={3}>

    <StepHikeCompact.Details title="连接">

    添加连接信息。前往您的 [`数据库设置`](https://supabase.com/dashboard/project/_/settings/database)。确保启用了 `使用连接池`。将连接模式切换为 `Session` 并复制您的连接参数。填入您创建项目时设置的数据库密码（如果您没有密码，可以在上面的数据库设置中重置）。

    </StepHikeCompact.Details>

    <StepHikeCompact.Code>

     ![添加连接信息。](/docs/img/guides/database/connecting-to-postgres/pgadmin/add-pg-server-conn-info.png)

    </StepHikeCompact.Code>

  </StepHikeCompact.Step>

  <StepHikeCompact.Step step={4}>

    <StepHikeCompact.Details title="SSL">

    从仪表板的 [`数据库设置`](https://supabase.com/dashboard/project/_/settings/database) 下载您的 SSL 证书。

    在 pgAdmin 中，导航到参数选项卡并选择连接参数为根证书（Root Certificate）。接下来导航到根证书输入框，它将打开一个文件选择器模态框。选择您之前下载的证书并保存服务器详细信息。现在 pgAdmin 应该能够通过 SSL 连接到您的 Postgres。

    </StepHikeCompact.Details>

    <StepHikeCompact.Code>

     ![添加连接信息。](/docs/img/guides/database/connecting-to-postgres/pgadmin/database-settings-host.png)

    </StepHikeCompact.Code>

  </StepHikeCompact.Step>

</StepHikeCompact>
