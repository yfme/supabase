每个 Supabase 项目都配备完整的 [Postgres](https://www.postgresql.org/) 数据库，这是一个免费开源的数据库，被认为是世界上最稳定和最先进的数据库之一。

### 其他功能

- Supabase 通过我们的 [Realtime Server](https://github.com/supabase/realtime) 为 Postgres 扩展了实时功能。
- 每个项目都是一个完整的 Postgres 数据库，具有 `postgres` 级别的访问权限。
- Supabase 管理您的数据库备份。
- 直接从 CSV 或 Excel 电子表格导入数据。

<Admonition type="note">

数据库备份**不包括**通过存储 API 存储的对象，因为数据库仅包含这些对象的元数据。恢复旧备份不会恢复自备份以来已删除的对象。

</Admonition>

## 术语

### Postgres 还是 PostgreSQL？

{/* supa-mdx-lint-disable-next-line Rule004ExcludeWords */}

PostgreSQL 数据库源自 POSTGRES 项目，这是 1986 年在加州大学伯克利分校编写的一个软件包。该软件包包含一种名为"PostQUEL"的查询语言。

1994 年，Postgres95 基于 POSTGRES 代码构建，添加了 SQL 语言解释器来替代 PostQUEL。
{/* supa-mdx-lint-disable-next-line Rule004ExcludeWords */}

最终，Postgres95 更名为 PostgreSQL，以反映其 SQL 查询功能。
此后，许多人称其为 Postgres，因为这样不容易混淆。Supabase 注重简洁，所以我们也称其为 Postgres。
