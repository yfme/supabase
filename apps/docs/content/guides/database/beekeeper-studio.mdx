[`Beekeeper Studio Community`](https://www.beekeeperstudio.io/get-community) 是一个免费的数据库交互 GUI 工具。

<StepHikeCompact>

  <StepHikeCompact.Step step={1}>

    <StepHikeCompact.Details title="创建新连接">

        在 Beekeeper 中，创建一个新的 Postgres 连接。
    </StepHikeCompact.Details>

    <StepHikeCompact.Code>

    ![Postgres 连接](/docs/img/guides/database/connecting-to-postgres/beekeeper-studio/new-connection.png)

    </StepHikeCompact.Code>

  </StepHikeCompact.Step>

  <StepHikeCompact.Step step={2}>

    <StepHikeCompact.Details title="获取连接凭证">

    从 [`数据库设置`](https://supabase.com/dashboard/project/_/settings/database) 获取您的连接凭证。您需要：
    - 主机地址（host）
    - 用户名（username）
    - 密码（password）
    - 端口（port）


    </StepHikeCompact.Details>

    <StepHikeCompact.Code>
        将您的凭证添加到 Beekeeper 的连接表单中

        ![凭证](/docs/img/guides/database/connecting-to-postgres/beekeeper-studio/beekeeper-credentials.png)

    </StepHikeCompact.Code>

  </StepHikeCompact.Step>

  <StepHikeCompact.Step step={3}>

    <StepHikeCompact.Details title="下载 SSL 证书">

    从仪表板的 [`数据库设置`](https://supabase.com/dashboard/project/_/settings/database) 下载您的 SSL 证书
    ![SSL](/docs/img/guides/database/connecting-to-postgres/beekeeper-studio/certificate.png)
    </StepHikeCompact.Details>

    <StepHikeCompact.Code>

        将您的 SSL 证书添加到连接表单中
        ![SSL](/docs/img/guides/database/connecting-to-postgres/beekeeper-studio/certificate-beekeeper.png)

    </StepHikeCompact.Code>

  </StepHikeCompact.Step>

    <StepHikeCompact.Step step={4}>

    <StepHikeCompact.Details title="测试并连接">

    测试您的连接，然后进行连接
    </StepHikeCompact.Details>

    <StepHikeCompact.Code>

        ![SSL](/docs/img/guides/database/connecting-to-postgres/beekeeper-studio/connect.png)

    </StepHikeCompact.Code>

  </StepHikeCompact.Step>

</StepHikeCompact>
