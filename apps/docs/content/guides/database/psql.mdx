[`psql`](https://www.postgresql.org/docs/current/app-psql.html) 是 Postgres 自带的命令行工具。

## 使用 SSL 连接

您应该尽可能使用 SSL 连接到数据库，以防止窃听和中间人攻击。

您可以从应用程序的仪表板获取连接信息和服务器根证书：

![连接信息和证书。](/docs/img/database/database-settings-ssl.png)

下载您的 [SSL 证书](#使用-ssl-连接) 到 `/path/to/prod-supabase.cer`。

查找您的连接设置。前往 [`数据库设置`](https://supabase.com/dashboard/project/_/settings/database)，确保勾选了 `使用连接池`。将连接模式更改为 `Session`，然后将参数复制到连接字符串中：

```shell
psql "sslmode=verify-full sslrootcert=/path/to/prod-supabase.cer host=[CLOUD_PROVIDER]-0-[REGION].pooler.supabase.com dbname=postgres user=postgres.[PROJECT_REF]"
```
